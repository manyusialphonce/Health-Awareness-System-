# Health-Awareness-System

**Health-Awareness-System** is a web-based application designed to provide comprehensive health information, awareness, and guidance to users. The system aims to educate people about various diseases, their symptoms, preventive measures, and general health tips to improve overall well-being.

---

## Features

- **Browse Diseases:** View a wide range of diseases with detailed information.  
- **Symptoms & Prevention:** Learn about symptoms, causes, and preventive measures for each disease.  
- **Search Functionality:** Quickly search for specific diseases or health topics.  
- **User-Friendly Interface:** Simple and intuitive navigation for all users.  
- **Responsive Design:** Works well on mobile, tablet, and desktop devices.  
- **Contact Form:** Users can send questions or feedback to administrators.  

---

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server Environment:** XAMPP (Apache + MySQL)  
- **Optional Libraries:** jQuery, FontAwesome (for icons)  

---

## Purpose of the Project

This project was created to:  

1. Increase **health education** awareness in the community.  
2. Help users understand **symptoms and preventive measures** for common diseases.  
3. Promote **healthy behaviors** and lifestyle practices.  
4. Provide a **functional PHP + MySQL web application** example for students and beginner developers.  
5. Serve as a foundation for more advanced health management systems.
